      Bulk mode

      The same 8-specialist FSM that answers a single address is run in batch over every NYC public school (1,992 from the DOE location dataset). Each row is scored on a transparent rubric — Sandy 2012 inundation, DEP stormwater scenarios, FloodNet sensor history, USGS Ida 2021 high-water marks, LiDAR micro-topography, NYC 311 flood complaints. Tier-1 and Tier-2 schools have their full Granite-reconciled cited paragraph pre-computed; Tier-3 paragraphs are generated on click.

      Source

      NYC DOE 2019-2020 School Point Locations (a3nt-yts4). Same data substrate as the address mode.
